HTML Head tag

The <head> element is a container for metadata (data about data) and is placed between the <html> tag and the <body> tag.

Metadata is data about the HTML document. Metadata is not displayed on borowser.

Metadata defines the document title, character set, styles, scripts, and other meta information.

The following tags can go inside the <head> element:

  1. <title>
  2. <meta>
  3. <script>
  4. <link>
  5. <base>
  6. <noscript>
  7. <style>
